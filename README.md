# Career Recommendation System

A web-based **career recommendation system** built with **Python**, **scikit-learn**, and **Flask**.  
Users can input their interests, skills, and personality traits using sliders, and the model predicts a suitable career path.

---

## Features

- Trained on a sample career dataset with interest/skill ratings
- Uses **Random Forest Classifier** for prediction
- Interactive **sliders** for user input (1–5 scale)
- Responsive **Bootstrap** web interface
- Displays recommended career instantly
- Handles rare career classes safely
